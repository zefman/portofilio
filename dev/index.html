ABOUT

Jozef Maxted

Hi, I'm Jozef, a creative full-stack web developer based in Brighton. I have been building websites since 2008.

I have a passion for web technoligies and enjoy using my skills to complete projects for high profile clients at The Unit.

As well as producing serious work for serious clients, I love experimenting with web technologies for creative purposes. Recently I have worked with chatbots, machine learning, the internet of things, and I am addicted to creating generative art (this site is perfect example).

If it can be built using web technologies, I can build it for you. If it can't, lets try anyway!
