The Grid

scss / includes / layout / grid

The base grid uses the css wizardry grid.

Each row is defined by an element with the .grid class. Directly followed by each column, defined as .grid__item, using a .width-xx/xx class to define its width. Every column needs to be separated by an in-line HTML comment.

The Grid: Desktop

The Grid: Desktop > Tablet

scss / includes / settings / variables

Dropping the grid at different breakpoints uses prefixed modifying width classes. These breakpoints are stored in a $breakpoints array in the variables file.

For example, adding .tablet-width-xx/xx to .grid__item will cause that column to change to its new width when the viewport is within a "tablet" size.

The Grid: Desktop > Tablet > Mobile

The same modifying class works for mobile, adding .mobile-width-xx/xx will cause it to change size at the defined "mobile" width.

Colours

scss / includes / settings / variables

Colours should be split into two sections, branding and helpers. All colours are included in the $site-colours array, helper classes are created using this array. For example, adding "brand-primary" : $brand-primary to this array, will give you access to:

  • .bg-brand-primary
  • .btn--brand-primary
  • .text-brand-primary
  • .text-link-brand-primary
